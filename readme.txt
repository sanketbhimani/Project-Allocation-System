Short intro about project
______________________________________________________________________________________________________________________________________________________

-The project is divided into three part
				Student side
				Faculty side
				Admin side
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------			
-Admin side
			-here admin can add edit and remove student, faculty members and project with details
			-here admin can edit and delete all faculty members project
			-Admin can change phases like team formation, choice filling and result showing phases
			-Admin can manually allocate project to any student
			-Here are also feature of master reset for use again
			-send mail to proper faculty member and student
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------			
-Faculty side
			-Here faculty members can add project
			-they can also delete and edit project which they have added
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
-student side
			-in team formation phase:
						-send request and accept and also cancel request to form team
			-in choice filling phase
						-fill choices for project
			-in result phase
						-They can just see result
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
